char * CStringEncode::toURL (   ) 
 

URL文字列に変換します.

現在のバイト配列をURL(の付いたあれね)にエンコードします。
エンコードできない場合はnullを返します。
半角スペースは "+" に変換します。
'半角英数字 [A-Za-z0-9] および *-. の記号は変換しません。

引数:
なし 
戻り値:
変換後の文字列
注意:
不要になったら戻り値で返ってきたポインタをCStringEncode::ReleaseBuff関数で破棄する必要がある
